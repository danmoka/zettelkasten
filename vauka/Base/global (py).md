# Оператор global

Позволяет читать и изменять глобальные переменные, т.е. те имена, которые лежат на верхнем уровне включающего модуля.

Ссылаться на глобальные переменные можно без оператора `global`

## Пример

```python
x = 5

def fun():
    global x
    x = 7

fun()
print(x) # 7
```
