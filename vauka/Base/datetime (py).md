# Модуль datetime.

Модуль для работы с датой и временем.
Поиск информации аналогичен поиску информации по модулю `math`.

## Пример

```python
from datetime import date

now = date.today()
birthday = date(2000, 10, 23)
age = now - birthday
print(age.days)
```
